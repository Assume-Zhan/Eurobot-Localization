# Eurobot-Localization
> Eurobot localization workspace for 2023

## Install
```bash=1
# Move in ws
sh install.sh
source ~/.bashrc
```

## Current launch file : 

- Setup
```bash=1
roslaunch eurobot_localization odometry_setup.launch
```

- Run
```bash=1
roslaunch eurobot_localization eurobot_localization.launch
```
